## Install

Installation via Bun:

```bash
bun add vot.js
```

Installation via NPM:

```bash
npm install vot.js
```

## Build

To build, you must have:

- [Bun](https://bun.sh/)

Don't forget to install the dependencies:

```bash
bun install
```

Run build:

```bash
bun build:bun
```
