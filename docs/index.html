Chaimu

Chaimu is an audio player that synchronizes audio with video.

Demo

Demo website

Install

Installation via Bun:

bun add vot.js

Installation via NPM:

npm install vot.js

Build

To build, you must have:

  • Bun

Don't forget to install the dependencies:

bun install

Run build:

bun build:bun
